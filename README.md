# DeadBeatSimulator

Simulate a dead beat dad on demand!

## Local Development

### Prerequisites

[Unity](https://unity.com/download) 2022.3 LTS

### How to Run

1. Open a terminal

2. Clone this repository with Git

    ```bash
    git clone https://github.com/JudasLilith/DeadBeatSimulator.git
    ```

3. Open Unity and go to the `Projects` tab

4. Click the `Add` button and select `Add project from disk`

5. Once `DeadBeatSimulator` appears in your project list, click on it to open it

6. Press the play button at the top to run the game
