# DeadBeatSimulator

*Simulate a dead beat dad on demand!*

You are a dead beat dad driving home from the local bar. You get pulled over by the cops to take a breathalyzer test, but since you refuse, the cops start chasing you just 5 minutes away from your home.

**Your Goal**: Escape the cops long enough to get home!

## How to Play

There are two options to use for the controller:

1. Wii remote + breathalyzer *(recommended)*

2. Keyboard + mouse

### Wii Remote + Breathalyzer

- **Move forward** - 1

- **Move backward** - 2

- **Turn** - Arrow keys

### Keyboard + Mouse

- **Move forward** - Page up / Fn + up arrow

- **Move backward** - Page down / Fn + down arrow

- **Turn** - Arrow keys

## Local Development

### Prerequisites

- Unity [Hub](https://unity.com/download)

- Unity [2022.3.62f3](https://unity.com/releases/editor/whats-new/2022.3.62f3)

### How to Run

1. Open the Unity Hub, and go to the `Projects` tab

2. Click the `New project` button

3. Select `2022.3.62f3` for the editor version, and then choose `Universal 2D` for the template

4. Give the project a name of your choosing, select a location for the project, and click `Create project`

5. Open a terminal once the project has finished loading, and clone this repository with Git

    ```bash
    git clone https://github.com/JudasLilith/DeadBeatSimulator.git
    ```

6. Open your file explorer, and then locate the `DeadBeatSimulator` directory of the cloned repository

7. Move the `DeadBeatSimulator` folder to the `Assets` folder of your newly created Unity project

8. Return to your open Unity project, wait for it to finish loading, and press the play button at the top to run the game
